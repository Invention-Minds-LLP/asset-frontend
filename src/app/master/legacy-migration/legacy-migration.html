Legacy Asset Migration

Onboard pre-existing audited assets without recomputing prior-year depreciation

Three migration modes available:
  • Granular — full per-asset history known (original cost, acc. dep at cut-off)
  • Proportional — only category/pool totals known; system pro-rates
  • Carry as New — no history derivable; treat as fresh asset on migration date
Engine never recomputes depreciation for periods before migration date.
