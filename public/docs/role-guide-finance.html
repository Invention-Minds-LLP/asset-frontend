Step-by-Step Workflows

Workflow 1: How to Run Batch Depreciation

  1. Navigate to Finance & Analytics > Batch Depreciation in the sidebar.
  2. Select the Depreciation Period (month/quarter/year).
  3. Choose the Depreciation Method if not already set per asset (Straight Line, Reducing Balance, etc.).
  4. Click Preview to see the calculated depreciation for all applicable assets.
  5. Review the preview: check asset count, total depreciation amount, and any warnings for assets with missing data.
  6. Click Run Batch Depreciation to process and commit the depreciation entries.
  7. Download the Depreciation Report for accounting records.
  8. The updated book values will reflect across all asset views and financial dashboards.

Workflow 2: How to Review TCO for an Asset

  1. Navigate to Finance & Analytics > Cost Analysis in the sidebar.
  2. Search for the asset by name, code, or category.
  3. The TCO view shows: Purchase Cost, Maintenance Cost, Insurance Cost, Depreciation, and Revenue Generated.
  4. Review the cost breakdown chart to understand where money is being spent.
  5. Compare the asset's TCO against similar assets using the comparison feature.
  6. Check the Cost Per Usage Hour metric to evaluate efficiency.
  7. Use the Decision Engine link to see whether the system recommends maintaining or replacing the asset.

Workflow 3: How to Use the CFO Dashboard

  1. Navigate to Finance & Analytics > CFO Dashboard in the sidebar.
  2. The dashboard displays: Total Asset Value, Monthly Depreciation, CapEx vs OpEx Split, Budget Utilization.
  3. Use the Date Range selector to view data for specific periods.
  4. Click on CapEx or OpEx sections to drill down into spending details.
  5. Review the Top Spending Departments chart for cost distribution.
  6. Check Upcoming Insurance Renewals and Warranty Expirations for proactive planning.
  7. Export the dashboard data using the Export button for board presentations.
Tip: Run Batch Depreciation at the end of each month before closing the books. Always preview first to catch any missing asset data. The CFO Dashboard auto-refreshes after depreciation runs.
